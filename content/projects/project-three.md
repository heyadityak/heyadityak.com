## Overview

A clean, modern portfolio theme built for Hugo static site generator. Features a responsive design, dark mode support, and is styled with Tailwind CSS.

## Features

- **Dark Mode** - System-aware dark mode with manual toggle
- **Responsive** - Mobile-first design that looks great on all devices
- **Fast** - Optimized for performance with minimal JavaScript
- **SEO Ready** - Built-in SEO optimization with Open Graph support
- **Accessible** - WCAG compliant with proper ARIA labels

## Customization

The theme is highly customizable through Hugo's configuration system. Colors, fonts, and layouts can all be adjusted without modifying the theme files directly.
